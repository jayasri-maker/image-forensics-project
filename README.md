
# Image Forensics Project

Image Metadata Extraction and Analysis System for Digital Forensic Investigation.

## 🚀 Live Demo

https://image-forensics-project-on91.onrender.com

## 🛠️ Technologies Used

- Python
- Flask
- Pillow
- HTML
- Render
- GitHub

## ✨ Features

- Extract EXIF metadata
- Generate MD5 and SHA-256 hashes
- Detect editing software information
- Identify GPS metadata
- Generate forensic summary reports
